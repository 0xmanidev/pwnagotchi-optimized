This is a fork of the [original pwnagotchi project](https://github.com/evilsocket/pwnagotchi). i made little changes like removing kali linux dependencies that make this unstable
updated the go compiler and installed aircrack that's it

**This fork does not support the original RPiZW. The board is obsolete and underpowered. Issues about the original RPiZW will be immediately closed.**
